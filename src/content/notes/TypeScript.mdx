# Correct Type Hints

Don't forget to install your types:

```shell
bun install @bun/types
```

And then, add them to your `tsconfig.json`:

```json title=tsconfig.json
{
  "compilerOptions": {
    "types": ["node", "bun"],
    "skipLibCheck": true
  }
}
```
