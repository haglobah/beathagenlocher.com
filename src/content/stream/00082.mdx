Today I learned that [[Just]] can finally run [recipes in parallel](just.systems/man/en/parallelism.html).

This is what this looks like:

```just
[parallel]
dev: astro bsky

astro:
    npm run dev

bsky:
    cd bsky-post-server && bun run dev
```

Sweet!
